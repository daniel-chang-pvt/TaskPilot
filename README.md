# 自动任务 / TaskPilot

TaskPilot 是一个轻量版 Android 自动化工具，设计目标是提供类似 MacroDroid 的核心能力，但界面和规则模型更简洁。

```text
App 名称：自动任务
项目名：TaskPilot
包名：com.danielchang.taskpilot
默认语言：中文
英文语言：TaskPilot
```

## 当前第一版功能

- 总开关：全局启用或停用自动任务。
- 单条规则开关：每条规则可单独启用或停用。
- 规则列表：显示所有规则。
- 添加、编辑、删除、复制规则。
- 手动运行规则。
- 分类/标签。
- 执行日志：记录每次执行成功、失败、条件不满足。
- 权限检查页面。
- 多语言结构：`values/strings.xml` 和 `values-en/strings.xml`。
- GitHub Actions 自动构建 release APK。

## 重要限制

Android 新版本限制很多系统能力，第一版中部分功能属于“已建模/已预留入口”，需要用户授予权限或后续继续增强。

- 打开/关闭 Wi-Fi：普通 App 不能直接开关 Wi-Fi，目前提供打开 Wi-Fi 设置页。
- 移动数据开关：普通 App 基本不能直接控制。
- 关闭某个 App：普通 App 不能可靠关闭其他 App，后续可能需要辅助功能。
- App 打开/关闭检测：后续需要使用情况访问权限或辅助功能增强。
- 音量键触发：后续需要辅助功能或前台服务增强。
- 通知触发：需要用户手动开启通知使用权。
- 短信/电话：需要敏感权限，自己安装可用，上架应用商店会受到严格限制。

## 构建 APK

项目已配置 GitHub Actions。推送到 GitHub 后进入：

```text
Actions -> Build APK
```

下载 artifact：

```text
TaskPilot-release-apk
```

解压后安装：

```text
app-release.apk
```
